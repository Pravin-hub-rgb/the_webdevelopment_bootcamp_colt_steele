Flexbox : The basics

What is it?

Flexbox is a one-dimensional layout method for laying out items in rows or columns.

It's New(ish)

Flexbox is a recent addition to CSS, included to address common layout frustrations.

Why 'flex'?

Flexbox allows us to distribute space dynamically across elements of an unknown size, hence the term 'flex'.

To use flex properties we have to put display as 'flex' on the parent container

In the flex model there are two axis 1- Main axis (goes from left to right by default) 2- Cross axis(top to bottom)

Responsive Design

The Problem

As mobile devices and tablets became widely available, developers had a problem...how do we create websites that look good on all screen sizes?

One Approach

Early on, it was common to create separate stylesheets for different devices, or even completely different websites for each size.

Enter Responsive

These days, we typically create ONE website and stylesheet that is able to respond to different device sizes and features.

We make websites responsive using media queries

Media queries allow us to modiy our styles depending on particular parameters like screen width or device type.

                        /* Exact width */
                        @media (width: 360px) {
                            div {
                                color: red;

                        }
                        
                        /* Minimum width */
                        @media (min-width: 500px) { \\ these properties will work min
                            div {
                                background: yellow;
                            }
                        }
                        
                        /* Maximum width */
                        @media (max-width: 800px) { \\ these property will work max up to 800 px
                            div {
                                border: 2px solid blue;
                            }
                        }
