Boolean Logic

MAKING DECISIONS WITH JAVASCRIPT

Comparison Operators

  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
  • == equality
  • != not equal
  • === strict equality
  • !== strict not-equality

These all return boolean values (true or false)

                   
                   
                    1>2 // false

equality (==) vs strict equaliy (===)

Double Equals (==)

  • Checks for equality of value, but not equality of type.
  • It coerces both values to the same type and then compares them.
  • This can lead to some unexpected results!
                          

                            1 == 1 //true

                            1 == "1" //true

                            // == WEIRDNESS
                            
                            'b' == 'c'; //false
                            
                            7 == '7'; //true
                            
                            0 == "" ; //true
                            
                            true == false; //false
                            
                            0 == false; //true
                            
                            null == undefined; //true

Triple Equals

Checks for equality of value and also equality of type.

                           

                            1 === '1'; //false

                            null === undefined; //false

not equal (!=) vs strict not-equal (!==)

not equal (!=)

This does not check equality of type

                           
        
                            1 != 2; //true

                            1 != '1'; //false

strict not-equal (!==)

Checks equality

                           
                           
                            1 !== '1'; //true

Most of the time you should use '===' and '!=='

Else

If nothing else was true, do this

                    

                        let rating = -99;

                            console.log("YOU ARE A SUPERSTAR!");
                        }
                        else if (rating === 2){

Nesting

We can nest conditionals inside conditionals

                    

                        let password = "cat dog";
                        if (password.length>=6){
                           if (password.indexOf(' ')!== -1){
                              console.log("Password cannot include spaces");
                           }
                           else {
                              console.log("Valid password!!")
                           }
                        }
                        else {
                            console.log("Password too short!");
                        }

AND (&&)

Both sides must be true, for the entire thing to be true

                        
    
                            1 <= 4 && 'a' === 'a'; // true
    
                            9 > 10 && 9 >= 9; // false
                            
                            'abc'.length === 3 && 1+1 === 4; // false                        
                        
                    
                        

                            if (password.length >= 6 && password.indexOf(' ')=== -1)
                            {
                                console.log("do this");
                            }

OR (||)

If one side is true, the entire thing is true

                        

                            let age = 76;

                            if (age < 6 || age >= 65) {
                                // run this
                            }
                            else {                            
                                console.log('That will be $10 please');
                            }
                            
                        
                    
                        

                            1 !== 1 || 10 === 10 // true

                            0 || undefined // false
