Array Methods

  • Push - add to end
  • Pop - remove from end
  • Shift - remove from start
  • Unshift - add to start

More Methods

  • concat - merge arrays
  • includes - look for value
  • indexOf - just like string.indexOf
  • join - creates a string from an array
  • reverse - reverses an array
  • slice - copies a portion on an array
  • splice - removes/replaces elements
  • sort - sorts an array

Push

add to end

                            

                                let members = ['harry','hermione'];
                                members.push('malfoy');
                                members //["harry","hermione","malfoy"]
                            
                        
                            

                                // Pushing multiple elements

                                colors.push('red','blue');

Pop

remove from end

                            

                                let members = ['harry','hermione'];
                                let popPerson = members.pop();
                                members //["harry"]
                                popPerson //"hermione"

Shift

remove from beginning

                            

                                let members = ['harry','hermione'];
                                let shiftPerson = members.shift();
                                members //["hermoine"]
                                shiftPerson //"harry"

Unshift

add to beginning

                            

                                let members = ['harry','hermione'];
                                members.unshift('malfoy');
                                members //[ "malfoy","harry","hermoine"]

Concat

merge two or more arrays

                            

                                const array1 = ['a', 'b', 'c'];
                                const array2 = ['d', 'e', 'f'];
                                const array3 = array1.concat(array2);
                                array3 //["a", "b", "c", "d", "e", "f"]

Includes

Boolean method which determine whether an array includes a certain value among its entries

                            

                                const pets = ['cat', 'dog', 'bat'];
                                pets.includes('cat'); //true

indexOf

returns the first index at which a given element can be found in the array, or -1 if it is not present

                            

                                const beasts = ['ant', 'bison', 'camel', 'duck', 'bison'];
                                beasts.indexOf('bison'); //1

Reverse

reverses an array in place. The first array element becomes the last, and the last array element becomes the first.

                            

                                const array1 = ['one', 'two', 'three'];
                                array1.reverse();
                                array1 //["three", "two", "one"]

Careful: reverse is destructive -- it changes the original array.

Slice

It is way to get a copy of a portion of an array.

                            

                                const animals = ['ant', 'bison', 'camel', 'duck', 'elephant'];
                                animals.slice(2); // expected output: Array ["camel", "duck", "elephant"]
                                
                                animals.slice(2, 4); // expected output: Array ["camel", "duck"]

Splice

Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

                            

                                const months = ['Jan','March','April','June'];
                                months.splice(1,0,'Feb');
                                months //['Jan','Feb','March','April','June']

Sort

sorts the elements of an array in place and returns the sorted array.

                            

                                let scores = [1,70,100,2500,9,-12,0,34]
                                scores.sort() //[-12,0,1,100,2500,34,70,9]

We can pass function to make it work

Reference Types & Equality Testing

Example:

                            

                                'hi' === 'hi' //true
                                ['hi'] === ['hi'] //false
                                [1] === [1] //false

So, in case of arrays we are not comparing content, instead it is comparing the references in memory 

                            

                                let luckyNum = 87;

                                let luckyNums = ['2','234'];

While storing array, it is actually storing the reference number not the actual content like in luckyNum case 

                            

                                let nums = [1,2];
                                let numsCopy = nums; //it is storing the address as nums
                                nums.push(3); //changed in nums
                                numsCopy; //[1,2,3]
                                nums === numsCopy //true

numsCopy is also changed, that's because they both are referencing to the same address.

Arrays + Const

As the variable is storing the address of an array instead of the actual content of an array.

SO, THE VALUES CAN CHANGE

AS LONG AS THE REFERENCE REMAINS THE SAME

                        

                            const myEggs = ['brown','brown'];
                            myEggs.push('purple');
                            myEggs[0] = 'green';
                            
                            myEggs = ['blue','pink']; //No! here we are changing the address as we are redeclaring variable with new array.
