JS Objects

OUR SECOND DATA STRUCTURE!

So, just like an array, objects are going to help us store multiple pieces of data together in some sort of structure. But unlike arrays, where whole point is to order our data. That's not the case with objects, instead our data is store using key-value pair

Objects

  • Objects are collections of properties.
  • properties are a key-value pair.
  • Rather than accessing data using an index, we use custom keys.

Property = Key + Value

Accessing Data Out of Objects

                    

                        const person = {
                            firstName : 'Mick',
                            lastName : 'Jagger'
                        }
                   
                        // Option:1
                        person['firstName']; //"Mick"

                        // Option:2                    
                        person.firstName; //"Mick"

VALID KEYS

All keys are converted to strings

* Except for Symbols.

                   
                   
                    const years = {
                        1999 : 'Good', 2020 : 'Bad'
                     }
                     years["1999"] //"Good"
                     const dumbdumb = {true: 'dat', null: 'hat'}
                     dumbdumb["true"] //dat
                     //Every key is turned into string whether it is boolean or null...

'.' syntax vs '[]' syntax

                       
                       
                        const vehicles = {
                            car: 'lambo',
                            bikeName: 'kawa'
                         }
                         vehicles.car //"lambo"
                         vehicles["car"] //"lambo"
                         vehicles[car] //car is not defined
                         //above it is expecting a variable name
                         vehicles['bike'+'Name'] //"kawa"

So, using '[ ]' we have ability to use a variable name

                           
                           
                            let birthYear = 2000;
                            const years = {
                               1999 : 'Not born',
                               2000 : 'born',
                               2001 : 'not born'
                            }
                            years.birthYear //undefined
                            years[birthYear] //'born'

Modifying Objects

                   
                   
                    const midterms = {danielle: 96, thomas: 78}
                    //now updating
                    midterms.thomas = 79;
                    midterms //{danielle: 96, thomas: 79}
                    //adding key
                    midterms.ezra = 88;
                    midterms //{danielle: 96, thomas: 79, ezra: 88}
