The Return Keyword

The return statement ends function execution AND specifies the value to be returned by that function

Return

Built-in methods return values when we call them. We can store those values:

                       
                       
                        const idx = ['a','b','c'].indexOf('b');
                        idx; //1

When we use return keyword in function we can capture a return value in a variable

                        

                            function add(x,y){
                                return x+y; // Return!
                              }
                              
                              const sum = add(10,16);
                              sum; //26
                              const answer = add(100,200);
                              answer; //300

Scope

Variable "visibility"

The location where a variable is defined dictates where we have access to that variable

Function Scope

Variable that are defined inside a function are scoped only inside that function and are not accessible out of that function.

                            

                                function helpMe(){
                                    let msg = "I'm on fire!";
                                    msg; // "I'm on fire"
                                  }
                                  msg; // Not Defined!

msg is scoped to the helpMe function

                            

                                let bird = 'mandarin duck';
                                function birdWatch(){
                                  let bird = 'golden pheasant';
                                  bird; //'golden pheasant'
                                }
                                bird; //'mandarin duck'

bird is scoped to birdWatch function

Block Scope

'{}' are block scope (if else statements and loops) except functions.

Variable defined inside a block are only accessible inside that block.

But, if we declare our variable using var keyword, our variable are scoped to function but not for block

                            

                                if(true){
                                    let msg = 'hi';
                                  }
                                  msg; //not defined
                            
                        
                            

                                if(true){
                                    var msg='hi';
                                  }
                                  msg; "hi"

Lexical Scope

The variables which defined in parent (outer) function are accessible to child (inner) function but the variable which are defined in child function are not accessible to the parent scope.

                            

                                function outer(){
                                    let hero = 'Black Panther';
                                  
                                    function inner(){
                                      let cryForHelp = `${hero}, please save me!`;
                                      console.log(cryForHelp);
                                    }
                                    inner();
                                  }

'this' in Methods

Use the keyword this to access other properties on the same object

We commonly use this inside of an object in a method and then we can access the value associated to key inside the object in a method

this refers to object inside which we use it.

                            

                                const person = {
                                    first: 'Robert',
                                    last: 'Herjavec',
                                    fullName() {
                                      return '${this.first} ${this.last}'
                                    }
                                  }
                                  person.fullName(); //"Robert Herjavec"
                                  person.last = "plant";
                                  person.fullName(); //"Robert Plant"

The value of this depends on the invocation context (the way we call the function) of the function it is used in.

                            

                                const cat={
                                    catName:"Snow bell",
                                    meow(){
                                      console.log(`${this.catName} said MEOW`)
                                      console.log(`${this}`)
                                    }
                                  }
                                  cat.meow() // 'this' refers to 'cat' object here
                                  //Snow bell said MEOW
                                  //[object Object]
                                  
                                  const func=cat.meow
                                  func() //here 'this' refers to 'Window' object
                                  //undefined said MEOW
                                  //[object Window]

Window object is the top level object inside which all other object resides.

And defined function becames its methods

                            

                                function ran(){
                                    console.log("yo! wassup wassup")
                                  }
                                  //ran function becomes method of Window object
                                  Window.ran() //yo! wassup wassup

Using TryCatch

Usually when there is error in our code, we get error message in console and the code which was after the error doesn't run. This can be solved using try catch

                        

                            try{
                                hello.toUpperCase();
                              } catch {
                                console.log("ERROR!!!!!!!");
                              } console.log("after")
                              //ERROR!!!!!!!
                              //after

We can also catch error

                        

                            function yell(msg){
                                try{
                                  console.log(msg.toUpperCase().repeat(3));
                                } catch (e){
                                  console.log("Please pass a string next time!");
                                }
                              }
                              yell("hi"); //HIHIHI
                              yell(234); //Please pass a string next time!
