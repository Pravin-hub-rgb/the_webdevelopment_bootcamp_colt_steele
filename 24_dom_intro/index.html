Web Development 2022

DOM (Document Object Model)

What is it?

  • The DOM is a JavaScript representation of a webpage.
  • It's your JS "window" into the contents of a webpage
  • It's just a bunch of objects that you can interact with via JS.

The Document Object

HTML & CSS goes in.... JavaScript object comes out

document > body > ul > li 

                   
                   
                    console.dir(document);

The document object is our entry point into the world of the DOM. It contains objects that represents all the content on a page, plus tons of useful methods and properties.

Selecting

Methods which help us select elements

  • getElementById
  • getElementsByTagName
  • getElementsByClassName

document.getElementById()

This method returns an element object representing the element whose id property matches the specified string.

<h1 id="heading">This is heading</h1>

const heading = document.getElementById('heading');

document.getElementsByTagName()

It returns an HTMLcollection of elements with the given tag name.

<p>this is 1 para</p>
<p>this is 2 para</p>
<p>this is 3 para</p>

const allPara = document.getElementsByTagName('p')

document.getElementsByClassName()

<img src="" alt="" class="square">
<img src="" alt="" class="square">
<img src="" alt="" class="square">

const allImg = document.getElementsByClassName('square')

querySelector

→ A newer, all-in-one method to select a single element.

                        

                            //Finds first h1 element:
                            document.querySelector('h1');
                            
                            //Finds first element with ID of red:
                            document.querySelector('#red');
                            
                            //Finds first element with class of big:
                            document.querySelector('.big');
                            
                            //Complex selection
                            document.querySelector('img:nth-of-type(2)');

querySelectorAll

Same idea as querySelector, but returns a collection of matching elements

                        

                            document.querySelectorAll('p a'); // gives all a tag and we can loop over them

So, basically we can use only querySelector & querySelectorAll

innerText

<h1>lololol</h1>

const h1 = document.querySelector('h1');
h1.innerText = "lololol";

textContent

it is also same as innerText but this shows all text no matter if it's display is set to none.

<p><span style="display: none;">hi</span>..hello</p>

const para = document.querySelector('p');
para.textContent; //hi..hello

innerHTML

Gets or sets the HTML markup contained within the element.

                                

                                    document.querySelector('p').innerHTML = ' this will be added ';
                                    //this will overwrite every thing (use += to not overwrite)

getAttribute()

Gives attribute of selected element.

                                

                                    document.querySelector('img').getAttribute('id'); 
                                    //we can also do this document.querySelector('img').src
                                    //we can also do this document.querySelector('img').alt

setAttribute()

                                

                                    document.querySelector('a').setAttribute('href','http://www.google.com') 
                                    // also this works - document.querySelector('a').href='http://www.google.com'

removeAttribute()

                                

                                    document.querySelector('button').removeAttribute('disabled')

style

Changes style of elements (all property name are camel case)

This style object doesn't contain style from our style sheet, instead it contains inline style.

                            

                                document.querySelector('h1').style.color = 'green'
                                 document.querySelector('h1').style.fontSize = '3rem'

                                // we can get computed CSS style using window object
                                window.getComputedStyle(h1).fontSize; //32px"
                                
                                //we can also do it using setAttribute
                                document.querySelector('h2').setAttribute('class','purple'); 
                                // all the style of the class get added to h2.

So, the better way to change the style is using classList

classList

It is an object by which we can control class and retrive and manipulate them.

                            

                                // classList.add
                                const h2 = document.querySelector('h2');
                                h2.classList.add('purple');
                                h2.classList.add('border');
                                
                                // we can remove class too
                                h2.classList.remove('border');
                                //checking if elements contains a class or not
                                h2.classList.contains('border'); //true
                                
                                // we can toggle the class also
                                h2.classList.toggle('border');

Creating a new element from scratch and append or prepending to the page

document.createElement()

This method creates the HTML element specified by tagName.

appendChild()

Put element after last child of parent 

                            

                                const img = document.createElement('img'); 
                                // this just create and now we have to add attributes like source.
                                img.setAttribute('src','https://blablalbl...') 
                                //this will still show on the page.
                                // now we have to append it to the page.
                                document.body.appendChild(img);

append()

This methods inserts a set of Node object or DOMString objects after the last child of parentNode.

                            

                                // we can also use it to insert text inside parent innerText
                                const p = document.querySelector('p');
                                p.append("i am new text yaaaaay!!");

prepend()

inserts as first child.

                            

                                const newPara = document.createElement('p');
                                newPara.append('text inside para');
                                document.body.prepend(newPara);

insertAdjacentElement()

Inserting adjacent to element.

Syntax - targetElement.insertAdjacentElement(position, element); 
                            

                                const h2 = document.createElement('h2');
                                h2.append('Are adorable chickens');
                                const h1 = document.createElement('h1');
                                h1.insertAdjacentElement('afterend',h2);
                                //same we can do with targetElement.after()
