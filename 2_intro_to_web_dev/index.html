Web Development 2022

Introduction to Web Development

The internet

  • What is it? → "A global network of networks." (It's just a bunch of connected computers.)
  • A router is a device that connects a computer system to many other computers.
  • The Internet is, in reality a network of networks.
  • The internet is the infrastructure that carries things like:
    1. Email
    2. The Web
    3. File Sharing
    4. Online Gaming
    5. Streaming Services.

The Web

  • The World Wide Web is an information system where documents and other resources are available over the Internet.
  • And those documents are identified using URLs
  • Documents are transferred via HTTP.
  • HTTP is a protocol that defines how forms of communication should work.
  • A web server is a computer that can satisfy requests on the web.

The Request/Response Cycle

  • Let's talk about what happens when that server response with something.
  • Once a browser makes a request and it reaches the server, the server does not respond with a complete web page; instead, it responds with instructions that the browser can understand and then display.
  • So the job of the browser is to display or render content for humans.
  • The server response code consists of HTML, CSS, and Javascript.

Front end and Back end

  • To give a response, there is code that has to run on the server itself to figure out what to respond with, and this is where the terms "front end" and "back end" come from.
  • The construction of responses, like figuring out what to say, who is asking, which websites are showing the results, what kind of ads to show, etc., is happening before sending the response, and that is happening on the back end.
  • And then the client-side code is sent to display the response on the front end.

What do HTML, CSS, and JavaScript do?

These are the three technologies that our browser understands and can utilize, and each of them has different responsibilities.

  • HTML is the underlying structure of a webpage.
  • CSS describes the structure and how it should look.
  • JS builds the logic for a webpage.
