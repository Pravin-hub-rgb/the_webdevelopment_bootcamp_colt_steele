CSS Selectors

CSS SELECTORS: A Super Important Part Of CSS

Universal Selector

Selects Everything!

                        

                            * {
                                color: black;
                            }

Element Selector

Select All Images

                        

                            img {
                                width: 100px;
                                height: 200px;
                            }

Combining Selectors

                        

                            h1, h2 {
                                color: magenta;
                            }

ID SELECTOR

Select the element with id of 'logout'

                        

                            #logout {
                                color: green;
                            }

There should be only one "id" in a single HTML file.

CLASS SELECTOR

Select elements with a class of "complete"

                        

                            .complete {
                                color: green;
                            }

class hook can be applied to many elements in a single HTML file.

DESCENDANT SELECTOR

Select all <a>'s that are nested inside an <li>

                        

                            li a {
                                color: teal;
                            }

So the element is somewhere inside the other element. (Direct child not necessarily)

ADJACENT SELECTOR

Select only the paragraphs that are immediately preceded by an <h1>

                        

                            h1 + p {
                                color:red;
                            }

paragraphs that come right after h1 on the same level.

DIRECT CHILD

Select only the <li>'s that are direct children of a <div> element.

                        

                            div > li {
                                color: white;
                            }

ATTRIBUTE SELECTOR

Select all input elements where the type attribute is set to "text."

                        

                            input[type="text"] {
                                color:yellow;
                            }
                                
                            a[href="http://www.google.com"] {
                                color:yellow;
                            }

PSEUDO CLASSES

Keyword added to a selector that specifies a special state of the selected element(s)

  • :active
  • :checked
  • :first
  • :first-child
  • :hover
  • :not()
  • :nth-child()
  • :nth-of-type()

When <a> is in the state of hover 

                            

                                a:hover {
                                    color:orange;
                                }

:active

                            

                                input:active {
                                    background-color:red;
                                }

PSEUDO ELEMENTS

Keyword added to a selector that lets you style a particular part of selected element(s)

  • ::after
  • ::before
  • ::first-letter
  • ::first-line

Specificity

Specificity is how the browser decides which rules to apply when multiple rules could apply to the same element.

It is a measure of how specific a given selector is. The more specific selector "wins".

General formula of specificity

INLINE STYLES > ID > CLASS > ELEMENT

The !important exception

When an important rule is used in a style declaration, this tells the browser that this is the most specific thing possible and should override any other declarations.

                    

                        h1 {
                            color: red !important;
                        }

Inheritance

If styles are applied to the parent element, they will be inherited by the child element if they are not applied specifically to the child element.

                    

                        body{
                            color: red;
                        }

The whole text inside the body element will turn red.

The Cascade

The order your styles are declared in and linked to matters!

                    

                        h1 {
                            color: red;
                        }
                    
                
                    

                        h1 {
                            color: purple;
                        }

The colour purple will be applied to h1.

Same with the style sheets.
