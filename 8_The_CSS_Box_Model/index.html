The Box Model

Width & Height

Border

border: 10px solid black;
box-sizing: content-box;
border: 10px solid black;
box-sizing: border-box;

Both boxes have given the width of 400px, but the box1 width becomes 420px because the width of the border is also added to it, while in box2, the width remains 400px because we have set box-sizing: border-box;

border-left-color:red;

border-left-width:2px;

Border-radius

border-radius:20px

border-radius:100%

border-radius:100% 20% 50% 10%;
border-left-style:dotted;

Padding

The space between the content box and the border.

Individual Properties - padding-left, padding-right, padding-bottom, padding-top

Shorthand Property - Set all four sides at once! padding:10px

Margin

So, margin is the space outside of border.

Individual Properties - margin-left, margin-right, margin-bottom, margin-top

Shorthand Property - Set all four sides at once! padding:10px

Display Property

Inline

Width & height are ignored. Margin & padding push elements away horizontally but not vertically.

Block

Block elements break the flow of a document. Width & height, margin, and padding are respected.

Inline-block

Behaves like an inline element, except width, height, margin, and padding are respected.

We can hide an element by setting display to none. display:none;

Percentages

Percentages are always relative to some other value.

Sometimes it's a value from the parent, and other times it's a value from the element itself.

width: 50%; - half the width of the parent

line-height: 50%; - half the font-size of the element itself.

em

EMs are relative units.

With font-size, 1em equals the font-size of the parent. 2em's is twice the font-size of the parent, etc.
With other properties (margin or padding), 1 em is equal to the computed font-size of the element itself.

If the parent element's font size is 30px, then 2em's font size will be 60px.p>

rem

Root ems

Relative to the root html element's font-size. Often easier to work with.

If the root font-size is 20 px, 1 rem is always 20 px, 2 rem is always 40 px, etc.
