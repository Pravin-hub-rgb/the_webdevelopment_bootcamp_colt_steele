The position CSS property sets how an element is positioned in a document, The top, right, botton and left properties determine the final location of positioned elements.

position:static

The element is positioned according to the normal flow of the document. The top, right, bottom, left and z-index properties have no effect. This is the default value.

test

position:relative

The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values of top, right, botton, and left. The offset does not affect the position of any other elements; thus, the space given for the element in the page layout is the same as if position were static.
Element is moving relative to the position where it would be normally.

position:absolute;

The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any; otherwise, it is placed relative to the initial containing block. It's final position is determined by values of top, right, bottom and left.

test





position:fixed;

The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to the initial containing block established by the viewport, except when one of its ancestors has a trnasform, perspective, or filter property set to something other than none, in which case that ancestor behaves as the containing block. Its final position is determined by the values of top, right, bottom, and left.
