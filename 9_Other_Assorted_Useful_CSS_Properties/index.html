The Alpha Channel

rgba(0,200,2,0.5)
there is the fourth channel which covers the transparency of the color.
It's value is between 0 to 1.

So, this is not going to affect the text color of the content inside it.

Opacity

opcaity: 0.5;
Opacity covers the transparency of entire element including it's content.

So, this is going to affect the text color of the content inside it.

Transform

The transform CSS property lets you rotate, scale, skew, or translate an element. It modifies the coordinate space of the CSS visual formatting model.

transform: rotate(40deg);
transform:rotateY(360deg)
transform:rotateX(360deg)
transform:scale(2);
transform:scaleX(2);
transform:scaleY(2);
transform:scale(2,0.3);
transform:translateX(50px);
transform:translateY(50px);
